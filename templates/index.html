What ADVRET does

ADVRET evaluates how vulnerable an image classification model is to adversarial attacks (FGSM, PGD, DeepFool, Carlini-Wagner). Upload a folder of images, select an attack mode, and ADVRET will generate static benchmark graphs and a downloadable report.

Inputs

  • - Pretrained model or custom .pt/.pth
  • - Folder of images (.jpg/.jpeg/.png)

Outputs

  • - Success rate & confidence-drop graphs
  • - Robustness curve
  • - Summary + defense suggestions

Instructions

  1. 1 Select or upload a model.
  2. 2 Select an image folder.
  3. 3 Choose attack mode, then run.
